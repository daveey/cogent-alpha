Beliefs & Learnings

  • Bug fixes > parameter tuning. The biggest wins came from fixing bugs (role misassignment, agent_id normalization, critic field mismatch) — not from tweaking thresholds.
  • Hub penalty helps. Despite alpha.0 not having it, removing hub penalty caused 2x regression. Agents without it wander to distant junctions and waste time.
  • Local scores don't predict tournament. v19 scored 18.38 local but only 2.76 in tournament. Cooperative team scoring amplifies strong play differently than self-play.
  • Seed variance is extreme. Scores range 0.00–12.29 across seeds. Single-seed optimization is noise. Need 5+ seeds minimum.
  • Agents capture junctions early but lose them. At step 1000: 8 friendly junctions. By step 3000: 0-2. Defense, not offense, is the bottleneck.
  • PCO produces correct signals after critic fix. Junction loss magnitude (925-1522) correctly identifies the main weakness. Resource and survival are fine.
  • PCO patches need API docs. Without GameState API reference, the learner invents non-existent methods. With docs, patches are clean and valid.
  • No shared state between agents. Agents are independent, may run in separate processes. Junction memory is per-agent only.
